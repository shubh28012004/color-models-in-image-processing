# color-models-in-image-processing
  RGB = 255,255,255 - White colour 
  RGB = 0,0,0 - Black color 
  0-black
  1-white
  
  RGB TO GRAYSCALE 
1. Lightness method 
    grayscale = (min(RGB) + max(RGB))/2
2. Average Method 
    grayscale = (R+G+B)/2
3. luminosity Method 
    grayscale = 0.3*R + 0.59*G + 0.11*B
